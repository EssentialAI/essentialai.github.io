Multiple Regression

In the previous article, we discussed a naive approach solve 1-dimensional linear regression. However, many times the target variable y^\hat{y}y^​ depends on more than one variable.

Variables for multiple regression:​

The input XXX is matrix of dimensions N×DN \times DN×D, where NNN is the number of data points and DDD is the number of independent features.

The feature vector www represents weights for every feature. www has the dimension, D×1D \times 1D×1.

The output y⃗\vec{y}y​ becomes a vector of dimensions N×1N \times 1N×1.

y⃗N×1=XN×DwD×1\begin{align} \vec{y}_{N\times1} = X^{}_{N\times D}w^{}_{D\times1} \end{align}y​N×1​=XN×D​wD×1​​​

Error function:​

The error function remains the same, the difference is that, we are calculating multiple weights at the same time.

E=∑i=1N(yi−yi^)2=∑i=1N(yi−wTxi)2\begin{align} E = \sum_{i=1}^{N}(y_{i}-\hat{y_{i}})^2 = \sum_{i=1}^{N}(y_{i}-w^Tx_{i})^2 \end{align}E=i=1∑N​(yi​−yi​^​)2=i=1∑N​(yi​−wTxi​)2​​

Summary​

This article explains multiple regression from scratch and provides a naive solution using calculus. However, one important detail here is that, in the above form (XTX)w=XTy(X^TX)w=X^Ty(XTX)w=XTy, for www to exist, (XTX)−1(X^TX)^{-1}(XTX)−1 must exist. This solution is inefficient for two reasons:

  1. Matrix inversion is difficult and power hungry.
  2. There can be cases where the inverse of XTXX^TXXTX does not exist. This means this method becomes unusable.

To overcome these issues, we use the Gradient Descent method to find the values of weights. Gradient Descent will be explained clearly in the next article.
